职位名称：高级后端工程师（基础设施）
工作地点：远程 / 西雅图
公司：Stripe / Uber 规模科技公司

关于该职位：
我们正在构建下一代全球金融基础设施。我们需要一位精通分布式系统的工程师，能够在睡梦中调试复杂的竞争条件，并深切关注大规模系统的可靠性。

职责：
- 使用 Go 和 Java 设计并实现高吞吐量、低延迟的服务。
- 为 PB 级持久化层（PostgreSQL、Cassandra）设计分片和复制策略。
- 改善系统可观测性，特别是在分布式追踪和指标聚合方面。
- 推动可靠性工程计划，包括混沌测试和速率限制策略，以防止级联故障。
- 指导高级工程师并定义 API 设计的工程标准（gRPC/Protobuf）。

要求：
- 在高增长环境中拥有 5 年以上分布式系统经验。
- 精通 Go 或 Rust。
- 深入理解数据库内部工作原理、并发模型和一致性保证（CAP 定理）。
- 具有大规模管理 Kubernetes 集群的经验（自定义控制器、自动扩展）。
- 拥有关键数据系统零停机迁移的成功记录。
